用each实现reduce/inject，用reduce/inject实现count，sum，product，map，select，reject，min，max，all?，any?

2016-08-01 functional ruby
reduce/inject，可带初始值，可在第二参数写方法名（运算符）

def reduce *args, &block
  acc, operator = args
  raise ArgumentError, 'both operator and actual block given' if operator && block
  raise ArgumentError, 'either operator or block must be given' unless operator || block

  block = block || -> (acc, value) { acc.send(operator, value) }

  each do |value|
    acc =
    if args.empty?
      args.push value
      value
    else
      block.call(acc, value)
    end
  end
  acc
end


测试：

count

def count
  inject(0){|c, e| c + 1}
end


sum

def sum from = 0
  inject(from, :+)
end


product

def product from = 1
  inject(from, :*)
end


map

def map &block
  inject([]) { |result, el| result + [block.call(el)] }
end


select

def select &block
  inject([]) { |result, el| result + (block.call(el) ? [el] : []) }
end


利用map和sum的话，select还可短一些：先map成二维数组（不符合条件的，二维为空数组），再用+拼接

def select(elements, &block)
  sum(map(elements) { block.call(el) ? [el] : [] })
end


reject可以是简单地减去select

def reject &block
  self - select(elements, &block)
end


min

def min &block
  inject(Float::INFINITY) do |minimum, el|
    el < minimum ? el : minimum
  end
end


max

def max &block
  inject(-Float::INFINITY) do |maximum, el|
    el > minimum ? el : maximum
  end
end


all?

def all? &block
  inject(true) { |result, el| result && block.call(el) }
end


any?

def any? &block
  inject(false) { |result, el| result || block.call(el) }
end
