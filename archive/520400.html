sinatra的extension与helper

2016-12-12 ruby sinatra versus
来自1.4.7

相同的是，都可用module或block来包装方法定义

区别在于，helpers是include，为Base添加实例方法，它可在处理块或模板内使用，因为处理块最终是bind到Application实例上执行的；而register是extend，为Base添加类方法，可当DSL使用

另外，register的module还可以定义一个名为registered的hook，在里面使用helpers

def helpers(*extensions, &block)
  class_eval(&block)   if block_given?
  include(*extensions) if extensions.any?
end

def register(*extensions, &block)
  extensions << Module.new(&block) if block_given?
  @extensions += extensions
  extensions.each do |extension|
    extend extension
    extension.registered(self) if extension.respond_to?(:registered)
  end
end
